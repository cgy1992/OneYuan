1. 以管理员身份点击运行install.bat脚本；
2. 运行冠苑打印终端管理系统软件即可。

卸载时已管理员身份点击运行uninstall.bat脚本

注意：安装完成后此文件夹不能删除。


配置文件说明：
config目录下有三个配置文件：
1. GYdataformatParam.xml:
此配置文件为xml格式，定义与银海的数据通信的报文格式。
此配置文件的目的是以后修改交互的报文格式时不用修改代码，只需要修改此文件即可。

按照约定，银海的数据都是以|符号分隔的属性值，属性包括数据源、卡号等信息。
配置文件中element即为一个属性，其各元素含义如下：
name: 报文格式中的属性含义，与银海提供的规范文档中的描述一一对应；
positionInIc: (xml文件中拼写错误，不用修改)此属性应该写在IC卡中的哪个位置，需要参考卡规范才能理解清楚；
isSendToIc: 此属性是否需要写IC卡，理论上都必须写，目前只有照片不需要写入；
isPrint: 此属性是否打印在社保卡上；
printName: 之前考虑扩展用，现在不适用，但不要删除；
required: 此属性是否允许为空，主要用于接受时报文合法性检查；
type:写在IC卡中的变量类型，也需要参考卡规范；
minlen：数据的最小长度，用于接受时报文合法性检查;
maxlen: 数据的最大长度，用于接受时报文合法性检查；

一般情况下，一个省市的报文格式约定好并调试完成后，此配置文件都不需要再修改。
只有在更换地市且银海修改报文格式后才需要修改。

注意：因为卡识别码的特殊性，报定义中必须要有"数据来源"这一属性，用于生成卡识别码。


2. GYprintParam.ini
此配置文件描述社保卡面上打印的信息位置，如姓名、卡号等。
目前只支持姓名、性别、卡号、社会保障卡号、发卡日期等的配置。
此配置文件不需要手动修改，通过软件上的卡片设置界面设置。

注意：系统启动时会检查此配置文件的isPrint的属性是否与报文格式的定义配置文件中的isPrint一致,
不一致则启动失败；

3. GYregParam.ini
此配置文件描述打印机及注册相关信息。
配置文件中除了强制写密钥参数需要后台手动修改外，其他各项都有配置界面一一对应。
几个不好理解的参数含义如下：
printerSam - slot: 打印机槽位，一般为槽位2，即获取加密因子的槽位；
shebaoSam - slot:社保卡槽位，一般为槽位３，即放加密卡的槽位；
shebaoSam - type:社保卡类型，目前有德阳的卡、甘孜州的卡，以及加密机方式；

当选择加密机方式时可以配置加密机的IP、端口、用户名和密码。

pin - check: 是否使用pin码加密
pin - pinCode: 加密用的pin码

cardReader - type: 读卡器类型：hd100等；
cardReader - debug: 目前不使用；

writeIc: 分别是匹配银行卡号写卡、匹配ATR值写卡、强制写密钥写卡。

一般情况下只有强制写密钥写卡需要后台修改。
